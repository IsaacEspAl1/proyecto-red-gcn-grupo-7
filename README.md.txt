# Proyecto de Investigación - Redes Neuronales de Grafos (GCN)

Este repositorio contiene un proyecto universitario para clasificación de nodos en una red social sintética usando Graph Convolutional Networks (GCN).

## Contenido

- Cuatro versiones de modelos GCN (`demos/`)
- Scripts para graficar precisión y pérdida (`plots/`)
- Resultados en CSV (`csv/`)
- Gráficos en PNG (`graficos/`)
- Dataset sintético en CSV
- Documento final del proyecto en PDF

## Requisitos

- Python 3.10+
- PyTorch
- PyTorch Geometric
- Pandas
- Matplotlib
- NetworkX

## Ejecución

1. Correr una demo:  
   `python demos/demo_basic.py`

2. Generar gráfico:  
   `python plots/plot_basic.py`
